# 멍톡 AI — 강아지가 말하는 통역기

10대·20대 여성 사용자에게 친숙하도록 파스텔톤, 말풍선 통역 결과, 감정 이모지 카드, 하단 네비게이션 중심으로 UI/UX를 재배치한 버전입니다.

## 주요 수정
- 통역 결과를 최상단 말풍선 카드로 배치
- 보호자/강아지 프로필 카드를 홈 상단에 추가
- 감정 확률을 귀여운 이모지 감정 카드로 표시
- 카메라 분석 수치를 숨기고 행동 칩으로 표현
- 버튼 문구를 `통역 시작하기`, `잠시 멈추기`, `통역 종료`로 변경
- 기술 수치, 모델 벡터, 일별 리포트는 사용자 화면에서 숨김 처리

## 실행
`index.html`을 브라우저에서 열면 됩니다. 마이크/카메라 권한을 허용하면 실시간 분석이 작동합니다.
