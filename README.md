# 멍톡 AI — 반응형 최적화 버전

반영 내용
- 바 타입 스마트폰: 하단 탭 바, 한 손 사용 카드 폭, 작은 화면 폰트/버튼 최적화
- 폴드·태블릿: 2열 카드 배치, 넓은 카메라/음성 그래프 카드 적용
- 컴퓨터: 좌측 세로 네비게이션 + 12컬럼 대시보드형 배치
- 강아지 이름 예시: 봄이
- 보호자 기본 호칭: 언니
- 카메라 켜기/끄기 전환
- 강아지 감지 박스, 꼬리 흔들림/주시/긴장/경계/화남 신호 퍼센트 시각화
- 화남/불만/짜증/소유욕/질투/강한 경고 언어 데이터 추가

실행 안내
- 카메라/마이크 기능은 브라우저 보안 정책상 HTTPS 또는 localhost 환경에서 가장 안정적으로 작동합니다.
- 실제 수의학적 진단 도구가 아니라 음성·행동 패턴 기반 추정 앱입니다.
