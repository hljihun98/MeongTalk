# 멍톡 AI - 음성 통역 중심 버전

카메라 기능을 제거하고, 음성 주파수 그래프 + 짖음 패턴 분석 + 성격 보정 + 귀여운 통역 UI 중심으로 정리한 버전입니다.

## 변경 사항
- 카메라 메뉴 및 카메라 분석 화면 제거
- 하단 메뉴를 홈 / 통역 / 설정 3개로 재정렬
- 통역 화면 아래에 음성 주파수 그래프 유지
- 성격 타입에 따른 음성 분석 보정 유지
- 모바일/태블릿/PC 반응형 레이아웃 유지

## 실행 방법
`index.html` 파일을 브라우저에서 열어 사용하세요.
마이크 권한이 필요하며, 일부 브라우저는 HTTPS 또는 localhost 환경에서만 마이크를 허용합니다.
