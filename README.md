# 멍톡 AI 반응형 수정본

## 이번 수정 사항
- 통역 결과 카드 안에 음성 주파수 그래프를 통합했습니다.
- 별도 음성 탭을 제거해 모바일 화면 흐름을 단순화했습니다.
- 카메라 켜기/끄기 버튼 전환을 유지하고, 전환 버튼은 카메라가 켜졌을 때만 작동하도록 개선했습니다.
- 카메라 권한 실패 시 원인 안내를 더 명확하게 표시합니다.
- 카메라 화면 위에 강아지 감지 박스, 얼굴/주시 표시, 꼬리/몸 움직임 시각화를 추가했습니다.
- 실제 웹캠 프레임 변화량을 읽어 움직임 점수를 계산하고, 화면 분석 퍼센트에 반영합니다.
- 강아지 감지/꼬리·몸 움직임/보호자 주시/긴장도/경계/화남 신호 퍼센트를 카메라 화면과 분석 카드에 표시합니다.

## 주의
브라우저 보안 정책상 카메라와 마이크는 HTTPS 또는 localhost 환경에서 가장 안정적으로 작동합니다.
이 앱의 카메라 분석은 웹캠 영상의 움직임과 사용자가 선택한 상황 보정값을 결합한 추정 시각화입니다. 실제 수의학 진단 도구가 아닙니다.
