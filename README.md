# 멍톡 AI — Voice Only Clean

이번 버전 변경사항
- 카메라 기능 제거 후 음성 기반 통역에 집중
- 성격 보정 기능 강화
  - 성격 타입: 보통 / 조용하고 예민 / 말 많고 활발 / 화가 많고 경계심 있음 / 애교 많음
  - 성격 보정 강도 0~100% 조절
  - 평소 하루 짖음 성향 반영
  - 기본 음성 추정 → 성격 보정 후 결과를 화면에 직접 표시
  - 관심/놀이/불안/경계/화남/스트레스 점수 실시간 표시
- 샘플 테스트 버튼으로 성격별 해석 차이 확인 가능

주의
- 이 앱은 Web Audio API 기반 음성 패턴 추정 프로그램입니다.
- 실제 수의학적 진단 도구가 아니며 반복적인 공격성, 통증, 호흡 이상은 전문가 상담이 필요합니다.


## 이번 수정
- '화가 많고 경계심 있는 친구' 성격에서 번역 말투가 더 단호하고 공격성 있게 나오도록 전용 문장 템플릿을 추가했습니다.
- 저음, 큰 소리, 거친 소리, 짧은 반복음에서 불만/짜증/강한 경고 가중치가 더 강하게 적용됩니다.
- 단, 실제 공격성 진단이 아니라 음성 패턴 기반 추정으로 표시됩니다.
