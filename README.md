# 멍톡 AI — 반응형/성격보정/카메라 추적 수정본

수정 사항
- 넓은 화면(PC)에서 카드가 한쪽으로 치우치던 그리드 배치를 재정렬했습니다.
- 음성 분석의 성격 보정값이 실제 최종 감정 판정에 적용되도록 확인했습니다.
- 카메라 화면에 MediaPipe 스타일 강아지 영역/얼굴/몸통/꼬리 추적 시각화를 추가했습니다.
- 강아지 감지, 꼬리·몸 움직임, 보호자 주시, 긴장도, 경계/화남 신호 퍼센트가 실시간 영상 변화 기반으로 갱신됩니다.
- 카메라가 꺼져 있을 때는 입력된 상황값으로 추정하고, 켜져 있을 때는 웹캠 영상 기반 움직임/밝기/대비 신호를 함께 반영합니다.

주의
- 브라우저 단일 HTML MVP이므로 실제 수의학적 진단이나 상용 AI 모델 판정이 아닙니다.
- 카메라는 HTTPS 또는 localhost에서 권한 허용 후 정상 작동합니다.
