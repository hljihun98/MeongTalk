# 멍톡 AI Named — 강아지가 말하는 통역 분석기

기존 Advanced 버전에 보호자 이름/강아지 이름 설정과 직접 화법 통역 문장 생성기를 추가한 버전입니다.

## 실행
1. 압축을 풉니다.
2. `index.html`을 브라우저에서 엽니다.
3. 보호자 이름과 강아지 이름을 입력하고 저장합니다.
4. `상황 샘플 주입` 또는 `실시간 분석 시작`을 누릅니다.

## 추가 기능
- 보호자 이름 저장
- 강아지 이름 저장
- “나랑 놀아줘 승연아!” 같은 강아지 직접 화법 출력
- 감정별 랜덤 문장 템플릿
- 이름 받침 여부에 따라 `아/야` 자동 처리
- 다른 말투로 다시 말하기
- 피드백 데이터에 보호자/강아지 이름 함께 저장

## 주의
이 앱은 수의학적 진단기가 아니라 음성·행동·상황 기반 추정 도구입니다. 통증, 공격성, 호흡 이상이 의심되면 수의사 상담이 우선입니다.
