# 멍톡 AI — 강아지 음성 패턴 통역기

## 수정 완료 사항
- 실제 파일명을 `index.html`로 정리하여 `manifest.json`의 `start_url`과 일치시켰습니다.
- README와 실제 코드가 맞도록 카메라 기능 및 카메라 관련 잔여 코드를 제거했습니다.
- Web Audio API 기반 음성 패턴 분석 중심으로 정리했습니다.
- 성격 타입, 보정 강도, 하루 짖음 성향, 상황 입력값을 반영하도록 유지했습니다.
- 서비스 워커에 기본 캐싱을 추가해 PWA 오프라인 지원을 개선했습니다.
- 앱 내부 문구에 “실제 진단 도구가 아님”을 명확히 표시했습니다.

## 실행 방법
1. 폴더 안의 `index.html`을 브라우저에서 엽니다.
2. 마이크 권한을 허용합니다.
3. 정확한 마이크 동작을 위해 배포 시 HTTPS 환경 또는 localhost 서버를 사용하세요.

예시:
```bash
python -m http.server 8000
```
브라우저에서 `http://localhost:8000` 접속

## 주의
이 앱은 강아지 소리의 크기, 길이, 주파수, 거친 정도를 규칙 기반으로 추정하는 재미/참고용 앱입니다. 실제 감정, 공격성, 통증, 질병을 진단하지 않습니다. 반복적인 공격성, 통증 의심, 호흡 이상, 식욕 저하가 있으면 수의사 또는 행동 전문가 상담을 권장합니다.
